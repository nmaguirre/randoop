      • contains

        public boolean contains(TypeArgument otherArgument)
        Checks whether this type argument contains another argument, using relationship defined in JLS Section 4.5.1. Checks for containment cases:
        • ? super T contains ? super S if S.isSubtypeOf(T)
        • ? super T contains ?
        • ? super T contains ? extends Object
        The last two being equivalent from the perspective of reflection.
        Specified by:
